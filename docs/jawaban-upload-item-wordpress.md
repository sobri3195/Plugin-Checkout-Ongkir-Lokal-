### HTML Description
Checkout Ongkir Lokal is a WooCommerce plugin built for Indonesian stores that need stable and reliable local shipping calculations at checkout.

It supports multi-courier rate logic, anti-downtime fallback strategies, COD risk controls, and operational tools for multi-origin fulfillment.

Core capabilities include:
- Shipping rate aggregation across multiple shipment origins.
- Rule-based COD decision support (allow/review/block signals).
- Packaging optimizer with volumetric weight calculation.
- Pickup point support on checkout.
- Address intelligence validation and normalization.
- Cost reconciliation between estimated and actual shipping cost.
- Event logging for troubleshooting and audit.

This plugin is designed for merchants who want to reduce checkout failures during carrier/API issues while keeping shipping operations controlled and measurable.
